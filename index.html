Tables TAGS
TAG Name Description
<table> table HTML tables allow web developers to arrange data into rows and columns
<th> table header Table headers
<tr> table rows Defines the rows of the tabel
<td> table data Where you insert the data in the tabel
<caption> caption Defines a table caption
<colgroup> colgroup Specifies a group of one or more columns in a table for formatting
<col> col Specifies column properties for each column within a <colgroup> element
<thead> thead Groups the header content in a table
<tbody> tbody Groups the body content in a table
<tfoot> tfoot Groups the footer content in a table
Basic TAGS
TAG Name Description
<!DOCTYPE> Document type Defines the document type
<html> HTML Defines an HTML document
<head> Head Contains metadata/information for the document
<title> Title Defines a title for the document
<body> Body Defines the document's body
<h1> to <h6> Headers Defines HTML headings
<p> Paragraph Defines a paragraph
<br> break Inserts a single line break
<hr> Thematic change Defines a thematic change in the content
<!--...--> Comments Defines a comment
Styles and Semantics TAGS
TAG Name Description
<style> Styles Defines style information for a document
<div> Div container Defines a section in a document
<span> Span area Defines a section in a document
<header> Header section Defines a header for a document or section
<footer> Footer Defines a footer for a document or section
<main> Main Specifies the main content of a document
<section> Section Defines a section in a document
<article> article Defines an article
<aside> Aside Defines content aside from the page content
<details> Details Defines additional details that the user can view or hide
<dialog> Dialog Defines a dialog box or window
<summary> Summary Defines a visible heading for a <details> element
<data> Data Adds a machine-readable translation of a given content
List TAGS
TAG Name Description
<ul> unordered list Defines an unordered list
<ol> ordered list Defines an ordered list
<li> li Defines a list item
<dl> description list Defines a description list
<dt> term name in the list Defines a term/name in a description list
<dd> term name of the list Defines a description of a term/name in a description list
